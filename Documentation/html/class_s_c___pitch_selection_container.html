Detailed Description

C# Class that contains the pitch and drum selections for a note in the Song Creation Interface.

Though called a "Pitch Selection Container", this class is also used for selecting drums. The drums and pitches are typecast to integer in the list of selected pitches anyway, so there is not much of a difference in how each are handled by this class.
