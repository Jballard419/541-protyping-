Handler for when a measure fills up.

Parameters
[in]aFullMeasureThe measure that filled up.
[in]aSpilloverHow much did the new note exceed the limit of the measure.
[in]aMelodyVelocityThe velocity of the pitches in the note.
[in]aLengthThe length of the note.
[in]aPitchesThe pitches in the note.
[in]aDrumVelocityThe velocity of the drums in the note.
[in]aDrumHitsThe drums that are hit for this note.
[in]aOffsetFromPrevNoteThe offset from the previous note.

This function creates a new measure and puts the note that couldn't be added to the last measure into the new one.
