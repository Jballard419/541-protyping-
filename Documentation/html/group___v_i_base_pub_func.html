float VirtualInstrument.GetAdjustedVelocityFactor ( int  aVelocity)

Gets the factor by which a note's waveform is scaled in order to account for velocity.

Parameters
[in]aVelocityThe velocity for the note from 0 (silent) to 100 (max volume).
Returns
A float between 0.0 and 1.0 where 0.0 means that the waveform should be made silent and 1.0 means that it should be unaltered.

Instruments without Built-In Dynamics just divide the given velocity by 100, but instruments with Built-in Dynamics map the factor to a ratio involving the Built-In Dynamics thresholds so that the volumes for velocities near a threshold don't have significant differences.

int VirtualInstrument.GetBuiltInDynamicsThresholdIndex ( int  aVelocity)

Gets the index for which sound file to use for a pitch at a given velocity.

Parameters
[in]aVelocityThe given velocity from 0 (silent) to 100 (max volume).
Returns
The index of the sound file to use.

For instruments with Built-In Dynamics (such as the Piano), this function takes a velocity and returns the index of the sound file to use for playing the sound. If a function does not support Built-In Dynamics, this function returns 0.

See also
Built-In Dynamics Thresholds.
