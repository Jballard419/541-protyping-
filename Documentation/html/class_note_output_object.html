Detailed Description

Unity GameObject that handles the actual output of sound.

This class handles the sound output for Songs and Notes. Copies of this object are dynamically created by the Virtual Instrument Manager for each note that can be played. Separate copies are also created to handle playing Songs and drum loops.
This class should only be handled by the Virtual Instrument Manager.
