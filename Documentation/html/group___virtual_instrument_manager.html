Detailed Description

Virtual Instrument Manager

Introduction

This class is a script for a Unity GameObject that manages a virtual instrument and its output. Its responsibilities include:

  • Initialization of the audio code
  • Serving as the bridge between non-audio objects and audio objects.
  • Connecting each NoteOutputObject which handles the output of audio to the VirtualInstrument which holds the audio data.
  • Mapping each Song to audio data from the VirtualInstrument and sending it to the proper NoteOutputObject.
  • Sending parameters for audio effects to the AudioMixer provided by Unity.
  • Handling Musical Typing Events.
  • Passing data to ATI_Diagnostics.
