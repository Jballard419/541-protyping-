Detailed Description

A specific instance of a VirtualInstrument that uses drum/cymbal samples.

Information

Currently, the drum kit supports playing 19 separate drums/cymbals. The kit only uses 13 sounds though, as the snare/tom sound files are duplicated for two entries each. This is done so that fast series of notes can be played on a tom or snare by spreading them across two keys rather than one.
Though there is an enum for drums, the base class deals in pitches. In order to account for this without having to duplicate a lot of code, the DrumKit actually acts on the pitch range from C0 to F1.
Currently, Built-In Dynamics are not supported for the DrumKit, but they will be in the future.
