Detailed Description

Used to load and store audio data for a specific instrument.

Information

Virtual instruments are responsible for loading the proper wav files, normalizing them, and providing the audio data from them to the Virtual Instrument Manager.

Supported Instruments

Right now we only support a Piano, Marimba, and Drum Kit.
