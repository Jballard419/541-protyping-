Detailed Description

A specific type of Virtual Instrument that uses piano samples.

The lowest pitch supported by the piano is B0 and the highest pitch supported is C8.


This instrument is special (so far) in this project in that it supports Built-In Dynamics with three different sound files for each pitch. Go to the section about Built-In Dynamics for more details.


Its wav files are stored in "Resources/Audio/VirtualInstrument/Piano/Samples/ff", "Resources/Audio/VirtualInstrument/Piano/Samples/mf", and "Resources/Audio/VirtualInstrument/Piano/Samples/pp"
