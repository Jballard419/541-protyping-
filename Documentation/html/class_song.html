Detailed Description

A C# class that represents a song.

This class's responsibilities include:

  • Serving as a container for a pre-arranged series of notes.
  • Converting the abstract representions of musical notes into a form that relates to the actual waveforms of the audio.
  • Saving and loading copies of itself to/from a file.
