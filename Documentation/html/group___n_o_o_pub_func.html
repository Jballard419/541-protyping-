void NoteOutputObject.BeingPlaying ( float  aVelocityFactor,
int  aDynamicsIndex,
int  aStartIndex = 0 
)

Handler for setting the note to be played. Should be called from the manager's PlayNote event.

Parameters
[in]aVelocityFactorThe adjusted velocity of the note to be played. Ranges from 0 (silent) to 1.0 (max volume).
[in]aDynamicsIndexthe index of the built-in dynamics.
[in]aStartIndexWhere the note/song/loop should begin playing. Defaults to 0.

This function signals to OnAudioFilterRead that it should begin playing audio. In order to avoid threading issues, the signalling mechanism functions somewhat like a Moore state machine where it'll have an effect on the next cycle of OnAudioFilterRead.
