void VirtualInstrumentManager.LoadNoteOutputObjects ( )
private

Loads each NoteOutputObject that is used to output the audio.

This function loads a NoteOutputObject for every currently playable note, and uses the loaded VirtualInstrument to set the audio data that they will output.
